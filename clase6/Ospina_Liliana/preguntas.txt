¿Quién inventó el sistema de control de versión Git y por qué?

El creador de Git es Linux Torvalds, hizo grandes aportes a lo que es la informática hoy en día.
La idea nació porque tuvo un problema con los desarrollos de software, trabajando con compañeros, detectó el problema de las distintas versiones, e inventó la solución de Git, la cual hace versiones de código.

¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018 Microsoft compró GitHub para enfocarse todo lo posible al desarrollo,
Fue comprado por la cantidad de 7500 millones de dólares, al inicio el cambio de propietario generó preocupaciones y la salida de algunos proyectos de este repositorio​, sin embargo no fueron representativos. 
GitHub continúa siendo la plataforma más importante de colaboración para proyectos Open Source
