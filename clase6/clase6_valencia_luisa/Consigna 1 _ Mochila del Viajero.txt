   1. ¿Quién inventó el sistema de control de versión Git y por qué?
Git fue creado por Linus Tordvalds con el propósito de llevar un control de versiones en proyectos que tenían archivos con un gran número de archivos de código fuente. Así logró llevar registro de los cambios en archivos locales y luego coordinar el trabajo de varias personas para compartir los archivos en un repositorio de código. 
   2. ¿A quién pertenece actualmente Github y por qué?
Actualmente Github le pertenece a Microsoft. Fue adquirida por esta gran empresa debido a la creciente utilización del software para diferentes proyectos.
